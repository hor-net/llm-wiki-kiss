# Indice del Wiki

Benvenuto nella knowledge base privata. Questa è la pagina radice del wiki.

## Sezioni principali

- [Progetti](projects/index.md) — documentazione dei progetti attivi.
- [Note](notes/index.md) — appunti, idee, osservazioni rapide.
- [Decisioni](decisions/index.md) — ADR e decisioni tecniche prese.
- [Riferimenti](references/index.md) — link, articoli, fonti esterne.
- [Log](logs/index.md) — log append-only di attività e eventi.

## Convenzioni

- I file sono in Markdown puro, UTF-8.
- I nomi delle pagine usano `kebab-case`.
- I link interni sono **relativi**: `[Test](../notes/altra-nota.md)`.
- Ogni pagina inizia con un titolo di primo livello (`#`).
- Nessun frontmatter obbligatorio: solo se serve metadata reale.

## Filosofia

- KISS prima di tutto.
- Il wiki conserva conoscenza stabile.
- La memoria conversazionale è gestita a parte (es. QMD).
- L'accesso avviene tramite server MCP e, opzionalmente, REST.
